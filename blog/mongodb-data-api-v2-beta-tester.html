MongoDB Data API v2 Beta Tester'ı olduk

Sizlere en güzel memeleri en hızlı biçimde sunabilmek için en son teknolojileri kullanıyoruz.

Sizlere sunduğumuz memeler buluttan geliyor, bulut derken bu bilişimdeki bulut sayın bakanımızın da belirttiği gibi:

İşte böyle bir bulut sistemi olan MongoLabs diye bir yerde tutuyoruz memeleri, bu bulut sisteminden meme getirmek için kullanacağımız API yetkilendirme desteği sunmuyordu. Yani bizim memeleri yüklediğimiz URL üzerinden kötü niyetli biri memeleri silebilirdi. İnanabiliyor musunuz! Bunu göze alamazdık, düşünsenize bir memeyi görüp bağlanıyorsunuz, ertesi gün heyecanla yeniden girip baktığınızda meme yok! Hayır buna izin veremezdik. Hemen MongoLabs ile yazışmalara başladık. Dedik böyle böyle bize salt okunur API verin falan, çok sinirliyiz bu arada, neyse yazdık döşedik yolladık. Bunlar da "tam bizde bunun üzerinde çalışıyorduk isteseniz size de açalım yeni APIyi kullanın bakın bakalım yapabilmiş miyiz" dediler.

Efendime söyliyim şu an dünyada bu yeni APIyi kullanan 9 sistemden birisi memeizle.com

Herşey size daha güzel memeleri daha hızlı sunabilmek için.
